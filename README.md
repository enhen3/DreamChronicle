## 夜梦录 DreamChronicle

记录、解读、洞见梦境的现代解梦应用（Vite + React + TypeScript + Tailwind + shadcn-ui）。

### 本地开发

```sh
pnpm i # 或 npm i / bun i / yarn
pnpm dev # 或 npm run dev
```

### 技术栈

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### 目录

- `src/pages/Index.tsx`：主页与交互逻辑
- `supabase/functions/interpret-dream`：云函数（解梦）
- `index.html`：站点元信息与 SEO
